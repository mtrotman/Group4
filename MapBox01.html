Baxter Creak Watershed Alliance

Exploring MapBox as a solution for the Baxter Creek Water Shed Alliance Website


Problem:

There are no good maps about the Baxter Creek Watershed. As a result, visitors to the area are not aware of the Baxter Creek Watershed and it's ecological or natural features.They may get lost whilst using the trails and miss key points of interest.


Solution:

To create a thematic map to display and educate the general public on the Baxter Creek Watershed, using MapBox.


What is MapBox?

MapBox is an online tool that gives people access to nagivation tools. Technology behind many apps that help people get to where they need to be. Thousands of companies utilize MapBox services in aid of navigational software in cars and smartphones. There are over 600 million monthly active users that trust their ability to run their services. This is a global service delivering to customers in farming, manufacturing, ecommerce and more industries.


Why use MapBox?

MapBox delivers results for any type of service needed. They are constantly updating their services to keep up with industry requirements. There are many options of Mapbox to fully supply any services needed of a client. The clientelle ranges from BMW to National Geographic to CNN.


Solution Steps:

Below are the steps we took to create the map featured on this page:


Step 1:

A group Github repository was created for Group4. All files used in this website are hosted there.


Step 2: MapBox Studio Manual - MapBox

The group created MapBox01.html using the MapBox Studio Manual using the MapBox GL JS featured.


Step 3: ArcGIS Pro - Problem - Feature Class to GeoJSON

We started bringing in files to ArcGIS Pro to set up the data as a GeoJSON file to bring into MapBox Editor. The original plan was to bring them in as GeoJSON files but that created an error within MapBox upon upload. We could not directly add data into MapBox Editor so we had to determine a work around to be able to upload the data.


Step 4: Converted MVT_Trails_Feature - Feature Class to Shapefile

The MVT_Trails_Features shape file was downloaded from ArcGIS Online and converted to shapefile after the GeoJSON upload technique failed. This was uploaded to the MapBox Custom Tilesets.

insert screen shot


Step 5: Add shapefiles within the Custom Tilesets to MapBox Editor

We added in the shapefiles to the MapBox Editor with success. We had to ZIP the files to bring them into MapBox. This was the workflow that finally allowed us to bring our data into MapBox!


Step 6: Bring Custom Tilesets into Outdoors Basemap

Added in new layers and connected the Tilesets to edit the map. The point symbols do not allow to change symbol - have to download symbols to further edit the design.


Step 7: Iframe

The zoom level needed to edited to allow the map to be properly visible within the Iframe. We used iframe to put one interactive map html into another webpage. We did this to better control the dimension of the map frame.
