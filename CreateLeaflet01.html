Baxter Creak Watershed Alliance

Exploring Leaflet as a solution for the Baxter Creek Water Shed Alliance website


Problem:

There are no good maps about the Baxter Creek Watershed. As a result, visitors to the area are not aware of the Baxter Creek Watershed and it's ecological or natural features.They may get lost whilst using the trails and miss key points of interest.


Solution:

To create a thematic map to display and educate the general public on the Baxter Creek Watershed, using Leaflet.


What is leaflet?

The map featured on this page was created using Leaflet. Leaflet is an open-source JavaScript library for mobile-friendly interactive maps. Leaflet features expansive libraries of third party plugins to choose from when creating your map. These range from but are not limited to tile & image layers, types of overlay displays and different types of user interfaces. These plug-ins are simple and effective to use.


Why use Leaflet?

Leaflet is open source, which means the original source code of Leaflet is made freely available and may be redistributed and modified. Secondly, Leaflet is light weight and mobile friendly. It works across a variety mobile and desktop platforms. Thirdly, the Leaflet API is well documented, with a focus on being easy to use. Lastly, there is a strong community behind Leaflet that makes it's useage possible.


Solution Steps

Below are the steps we took to create the map featured on this page:


Step 1

A group Github repository was created for Group4. All files used in this website are hosted there.


Step 2

The group created CreateLeaflet.html using the Quick Start Guide featured on the Leaflet Website: https://leafletjs.com/SlavaUkraini/examples/quick-start/. The Esri.WorldStreetMap tile layer from https://leaflet-extras.github.io/leaflet-providers/preview/ was selected however, it did not show. Thus, a tile layer from open street maps was chosen.

Step 3

Converted trails shape file to .geojason using ArcGISPro using the Features to GeoJSON conversion tool. Loaded this to Group 4 Github repository.

Step 4

Used geojson.io to create hypothetical fishing points in the Baxter Creek Water Shed. Saved this file to Grpup 4 Github repository.

Step 5

Experienced a bit of difficult trying to format the map size.There seemed to be conflict between with custom css file. Percentage sizing did not work whilst fixed pixel sizings did.

Step 6

We used iframe to intgrate map into this webpage as it was an easier solution.
